# Flashcards

This program allows users to set up a series of flashcards to test their 
memory recall.

This was created by completing the 
[Flashcards (Python) Project](https://hyperskill.org/projects/127) on 
JetBrains Academy.

## How to Use

When the program is run, the user is presented with the following options:

### Add

Adds a flashcard to the set. The program asks the user for the term being 
defined, and then a definition.

### Remove

Removes a flashcard from the set by its term.

### Import

Imports a set of flashcards from a CSV file.

### Export

Exports the set of flashcards in memory to a file.

### Ask

Quizzes the user on the flashcards in memory. The user can specify how many 
questions they want to be asked, and the program will loop through the list 
of flashcards until the desired number has been tested.

### Exit

Quits the program.

### Log

Logs the console output to a file.

### Hardest Card

Tells the user which card they had the most errors answering.

### Reset Stats

Resets the testing statistics.
